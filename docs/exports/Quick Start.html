§6 · FAQ 1/2

常見問題 1/2

Q 5 分鐘了狀態沒變?
A 系統 1 分鐘輪詢、最多等 2 分鐘。仍卡 → 找 Kay
Q 字太長會怎樣?
A 系統自動縮字級。建議大標 ≤ 12 / 小標 ≤ 15 / CTA ≤ 5
Q 商品圖被裁掉?
A 點「重做底圖」3 次仍不行 → 找 Kay 看 prompt
§6 · FAQ 2/2

常見問題 2/2

Q 字色看不清楚?
A 手動指定 🔵S3 大標色 + 描邊 → 重做成品
Q 一次開很多筆?
A 可以、系統排隊。建議單批 ≤ 5 筆
Q 急件怎麼辦?
A 標題加「[急]」+ Chat 私訊 Kay
